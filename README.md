# CodeQuest

A gamified Python + Data Structures & Algorithms learning app. 256 levels: a full Python basics curriculum (variables through decorators and OOP) plus 150 verified DSA problems (NeetCode-150 style), each with SVG diagrams, a runnable worked example, a free-practice stage, and a graded quiz that awards coins/XP and unlocks the next topic.

Live at: https://292akhil2929-cmyk.github.io/codequest/

## Stack

- React + Vite
- [Pyodide](https://pyodide.org/) (Python running in-browser via WebAssembly, in a Web Worker) for grading code
- Progress is stored in `localStorage` — no backend

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build     # outputs to dist/
npm run deploy     # builds and publishes dist/ to the gh-pages branch
```

## Content generation

This app's source (`src/utils/levels.js` and most of `src/`) is generated by a Python pipeline (not included in this repo) that writes and verifies every challenge by actually executing a reference solution locally before emitting it — so every expected test output is guaranteed correct, not hand-typed.
